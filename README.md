# paleofetch-artix
A fork of paleofetch suposed to be used on Artix Linux.

If you're on Arch, use the original:
https://github.com/sam-barr/paleofetch


I just changed the ascii logos and got rid of the battery information

## how to make
````bash
make install
````

## how to use
````
paleofetch
````
